# Installing Workflow package

To install the Workflow package in your Zango application, follow these steps:

#### 1. Access the App Panel

Navigate to your Zango application's landing page and access the App Panel.

#### 2. Navigate to the Packages Menu

In the App Panel, locate and click on the "Packages" menu.

#### 3. Install CRUD Package

You will see a list of available packages. Ensure the "CRUD" package is installed. If it's not installed, proceed to [CRUD Installation](https://docs.zelthy.com/crud) before moving forward.

#### 4. Install Workflow Package

Now, locate the "Workflow" package from the list.

#### 5. Initiate Installation

Click on the three-dot menu icon located on the right side of the "Workflow" package row. From the context menu that appears, select the "Install Package" option.

#### 6. Select Package Version

In the install package form, select the desired version of the Workflow package that you want to install.

#### 7. Confirm Installation

After selecting the version, click on the "Install Package" button to confirm the installation.

#### 8. Installation Progress

The page will load to indicate that the package is being installed. Wait for the installation process to complete.

#### 9. Verify Installation

Once installed, the status of the Workflow package should change to "Installed", and the selected version number will be displayed in the table.

By following these steps, you can successfully install the Workflow package in your Zango application, enabling its functionalities and features within your app.
